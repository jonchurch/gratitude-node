What is a Sheet?

We call "Sheet" to every page of this template, for example you have seen the "Documentation" sheet, and now you're watching the "Sheets" sheet.

Every sheet has a button in the right-side menu.

How to create a Sheet?

The main structure of this template is a container div with the class td-container. Inside you have the td-sheets-container div, there's where you are going to create your sheets!

The structure of a Sheet is the following:

<div class="td-sheet">
    <div class="td-sheet-icon"></div>
    <div class="td-sheet-title"></div>
</div>

3 Elements:

  • "TD-SHEET" contains all the code of the sheet.
  • "TD-SHEET-ICON" goes first inside, and inside this you add the code of the icon for that sheet in the menu.
  • "TD-SHEET-TITLE" in second place. Inside this div add the title of the sheet that will appear on the menu.

After the Title, you can add the remaining code of the sheet and content in whatever way you like.

Navigation

You can use the right menu to navigate through the sheets, or use the keyboard arrows.

Icons

This theme is completly compatible with external resources, so you might use any Icon Font of your choice.

There are 3 types of Icons Fonts included already, Bootstrap Glyphicons, Themify Icons, and Font Awesome

Bootstrap

This template is fully compatible with the last version of Bootstrap, so you might use it without any problems.

Services

To create a Services section with icons and descriptions, you have to follow the next steps:

For each item, you need to add a div with the class td-aboutus, containing another div with the class td-aboutus-header and inside it, a span with the font-icon, and after it a H3 with its title. After "td-aboutus-header" you'll add a P element with the description.

Animations

Animations

Animate.CSS plugin has been chosen as the animation source. In its website you'll find all the animations.

In order to add an animation to an element, you have to add data-animation="fadeIn" (change "fadein" for whatever animation you want to use).

If you wish to delay the animation, you can also add to the element data-animation-delay="300" with that number being the delay time in miliseconds.
